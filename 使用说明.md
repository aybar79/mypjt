# CS:GO 皮肤价格查询器 - 使用说明

## 🎯 项目概述

这是一个功能完整的CS:GO皮肤价格查询网站，具备以下核心功能：

✅ **实时价格查询** - 从Steam市场获取最新价格数据  
✅ **智能搜索** - 支持模糊搜索和自动完成  
✅ **价格历史** - 存储和展示历史价格变化  
✅ **多平台爬虫** - 集成多个交易网站数据源  
✅ **可视化图表** - 价格趋势图表展示  
✅ **响应式界面** - 现代化的用户界面  

## 🚀 快速启动

### 方式一：简化版服务器（推荐）
```bash
# 进入项目目录
cd /home/cg-42/myproject

# 启动简化版服务器（只需Python标准库）
python3 simple_server.py
```

### 方式二：完整版Flask服务器
```bash
# 安装依赖（需要pip）
pip3 install -r requirements.txt

# 启动Flask服务器
python3 app.py
```

## 🌐 访问网站

服务器启动后，在浏览器中访问：
- **简化版**: http://localhost:8000
- **完整版**: http://localhost:5000

## 📖 功能说明

### 1. 皮肤搜索
- 在搜索框中输入皮肤名称
- 支持中文和英文搜索
- 自动提供搜索建议
- 示例搜索词：
  - `AK-47 | Redline`
  - `AWP | Dragon Lore`
  - `M4A4 | Howl`

### 2. 价格信息
- **最低价格**: 当前市场最低售价
- **中位价格**: 市场中位价格
- **交易量**: 24小时交易数量
- **数据源**: 价格数据来源平台

### 3. 价格图表
- 显示最近30天价格趋势
- 交互式图表，支持悬停查看详情
- 自动更新历史数据

### 4. 多平台数据
支持的交易平台：
- **Steam Market** - Steam官方市场
- **Buff163** - 国内最大CS:GO交易平台
- **CS.Money** - 国际知名交易平台
- **BitSkins** - 老牌皮肤交易网站

## 🔍 故障排除

### 1. 端口占用
如果端口被占用，可以修改代码中的PORT变量：
```python
PORT = 8001  # 改为其他端口
```

### 2. 价格查询超时
- 检查网络连接
- Steam API可能有访问限制
- 等待一段时间后重试

### 3. 依赖安装失败
简化版服务器只需要Python标准库，无需安装额外依赖。

## 💡 使用技巧

1. **精确搜索**: 使用完整的皮肤名称获得最佳结果
2. **品质后缀**: 包含品质信息，如"(Field-Tested)"
3. **英文搜索**: Steam市场使用英文名称，建议使用英文搜索
4. **耐心等待**: 首次查询可能需要较长时间建立数据
